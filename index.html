一、border-radius 的使用

border-radius 给元素添加圆角边框，这是一个简写属性，是border-top-right-radius、border-top-right-radius、border-bottom-left-radius、border-bottom-right-radius简写的一个属性。

语法

语法解释

border-radius是由border-top-left-radius、border-top-right-radius、border-bottom-left-radius、border-bottom-right-radius简写成的，这个四个属性分别对应元素的四个角，同时每个角有两个方向，所以border-radius最多可以取8个值。根据CSS的取值顺序‘top - right - bottom -left’。
设置一个值时，所有的值都相同；
设置两个值：top bottom相同，left right相同；
设置三个值：top （left right相同） bottom；
设置四个值：top left right bottom；
border-radius 中出现 ‘/’ 为 水平半径 / 垂直半径。

二、border-radius 是如何画圆角的

top-left为例，设置50px的圆角，是从元素top-left的顶点定义为（0，0）然后把圆心平移到（50，50）画圆，所得到的边框。如果top-left设置的值不同，同样平移到坐标点，则画椭圆。其他top-right、bottom-left、bottom-right同理。

box-shadow的使用

发表于 2020-04-18 | 分类于 前端

一、CSS3 box-shadow属性介绍

box-shadow属性可以给页面元素添加阴影效果

语法

值 描述
offset-x 必需的。水平阴影的位置。允许为负值。
offset-y 必需的。垂直阴影的位置。允许为负值。
blur 可选。阴影模糊距离，即模糊程度。值越大，模糊面积越大，阴影就越大越淡。不能为负值。默认为0。
spread 可选。阴影的大小，即阴影的扩展长度值。取正值，阴影扩大；取负值阴影收缩，默认值0，阴影和元素同样大。
color 可选。阴影的颜色。
position 可选。唯一的值 【inset 内阴影】。

JavaScript语法

注：boxShadow属性可以定义多个阴影，每个阴影使用逗号隔开。
offset-x 设置水平阴影偏移量，如果为负值阴影位于元素的左边。
offset-y 设置垂直阴影偏移量，如果为负值阴影位于元素的上方。

二、box-shadow的使用

1、offset-x 的取值
box-shadow: 10px 0 0 0 red; box-shadow: -10px 0 0 0 red;

2、offset-y 的取值
box-shadow: 0 10px 0 0 red; box-shadow: 0 -10px 0 0 red;

3、blur 的取值

box-shadow: 0 10px 10px 0 red; 
注：这里是不是很奇怪为什么只设置了，offset-y的位置偏移，x方向上现在为什么也有阴影呢。看下面的例子，把元素的阴影从元素下方移出来。
box-shadow: 0 210px 0 0 red;

阴影移出来后，阴影的大小和元素的大小是相同，blur取正整数时，值越大，阴影面积就越大，顾超出了元素的大小。

4、spread 的取值
box-shadow: 0 210px 0 10px red; box-shadow: 0 210px 0 -10px red;

5、position

box-shadow: 0 10px 10px 0 red inset; 
注：position设置阴影的方向，不设置为外阴影；inset为内阴影。

三、box-shadow 干点啥

样式 效果
box-shadow: 23px 23px 0px -20px red, -23px -23px 0 -20px red;
box-shadow: 0px 0px 0px 1px red;(模拟边框)
box-shadow: 0px 0px 0px 5px red,0px 0px 0px 10px black,0px 0px 0px 15px gold;(多重边框)
