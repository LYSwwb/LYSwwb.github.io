一、什么是LINQ

LINQ 是 Language Integrated Query (语言集成查询)的缩写：LINQ，在.NET Framework 3.5、C# 3.0 时被引入。在传统上，对数据查询我们需要对每种不同的数据源，学习不同的查询语言，比如：SQL查询、XML查询等，采用LINQ，将帮我们更加方便的查询各种数据源。LINQ主要包括：LINQ to Objects、LINQ to Entities、LINQ to XML等

二、前期知识

1、隐式类型（var）

隐式类型的声明，可以采用var关键字，var 是从Visual C#3.0开LINQ始，在方法范围内声明的变量可以具有隐式“类型”var。隐式类型本地变量为强类型，就像用户已经自行声明该类型，但编译器决定类型一样。
这句话咱们换种说法就是：由于C#是强类型的语言，我们可以在方法范围内，使用隐式声明变量的类型 var， 编译器会通过初始化语句表达式推断出变量的类型。

注：C#为强类型语言，所以在使用var声明变量时，必须赋初始值，并且不能为null。编辑器会根据初始化表达式，决定变量的类型。

2、匿名类型

匿名类型提供了一个方便的方法，可用来将一组只读属性封装到单个对象中，而无需首先显示定义一个类型。类型名由编译器生成，并且不能再源代码级使用。每个属性的类型由编译器推断。可以使用new运算符和对象初始值创建匿名类型。匿名类型是直接从对象派生的类类型。

匿名类型通常用在查询表达式的select子句中，以便返回源序列中每个对象的属性子集。匿名类型所含的属性是公共只读属性，并且不包含其他种类的类成员。初始化属性的表达式不能为null、匿名函数。

匿名类型重写了 Equals 和 GetHashCode ，因此仅当同一个匿名类型的两个实例的所有属性都相等时，这两个实例才相等。

3、扩展方法

扩展方法可以让你可以向现有类型中“添加”方法，而无需创建新的派生类型，重新编译或者修改原始类型。扩展方法是一种特殊的静态方法，但是他们的调用就像是扩展类型上的实例方法一样。

我们在写LINQ代码的时候，将会经常用到扩展方法（LINQ标准查询运算符）。这些查询功能可以添加到现有的System.Collections.IEnumerable 和 System.Collections.Generic.IEnumerable类型。

扩展方法的定义：
1、定义包含扩展方法的静态类。该类必须对客户端代码可见。
2、将扩展方法实现为静态方法，并且使其可见性至少与所在类的可见性相同。
3、此方法的第一个参数指定方法所操作的类型；此参数前面必须加上this修饰符。
4、在调用代码中,添加using指令，用于指定包含扩展方法类的命名空间。
5、按照调用类型上的实例方法一样调用扩展方法。

Iterators 学习

首先咱们来看看什么是迭代，迭代是重复反馈过程的活动，其目的通常是为了逼近所需目标或结果。每一个次对过程的重复称为一次“迭代”，而每一次迭代得到的结果会作为下一次迭代的初始值。

在计算机程序中：迭代是程序中对一组指令（或一定步骤）的重复。而迭代器（Iterator）就是一个封装了迭代的对象。

对给定的list或tuple，我们可以通过for循环来遍历这个list或tuple，这种遍历我们称为迭代（Iteration）。

foreach遍历数组：下面看一个简单的foreach循环，遍历一个整数数组，并将个每个整数打印输出到控制台：

C#编译器会把foreach，转换成for循环实现。通过借助Length属性和索引运算符，C#编辑器可以使用for循环迭代数组中的每个元素。

foreach 和 IEnumerable集合：虽然前面的代码适用于长度固定且始终支持索引运算符的数组，但并不是所有类型集合的元素数量都是已知的。此外，许多集合类（包括Stack、Queue和Dictionary）都不支持按索引检索元素。因此，需要使用一种更为通用的方法来迭代元素集合。迭代器模式就派上用场了。假设可以确定第一个、第二个和最后一个元素，那么就没有必要知道元素数量，也没有必要支持按索引检索元素。

在编写代码过程中，我们经常对集合进行迭代，遍历集合中的每一项。
我们还将创建迭代器方法，这些方法将为该类的元素生成要给迭代器。迭代器可用于:
1、对集合中的每个项目执行操作；
2、枚举自定义集合；
3、扩展LINQ或其他库；
4、创建一个数据管道，数据通过迭代器方法有效地流动。

迭代器是枚举器的生成器。

yield return 语句返回该枚举器的下一个元素。

迭代器语义
迭代器是一个包含一个或多个yield 语句的方法，属性或索引器。

允许多个yield语句。

yield break

yield break 语句指示迭代器块应提早退出，而不返回更多元素。

返回语句在迭代器块中是非法的，请使用yield break.
