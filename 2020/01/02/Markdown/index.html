Markdown使用记录

发表于 2020-01-02 | 分类于 工具

Markdown 是一种轻量级标记语言，创始人：约翰.格鲁伯（John Gruber）。它允许使用简单方便书写的文本格式编写文档，并能转换成HTML文档。

Markdown 语法介绍

标题

Markdown 在文本前面使用“#”创建标题，共支持六级标题。

段落

在文字后方添加两个空格或者一个空行，表示换行。

字体属性

使用*加粗文字；

使用_倾斜文字

分隔线

可以使用星号、减号、底线创建分割线。

删除线

~~创建删除线

脚注

列表

有序列表

有序列表使用数字并加上. 号来表示

无序列表

无序列表使用星号(*)、加号(+)或是减号(-)作为列表标记

嵌套列表

区块

在段落开头使用 > 符号，创建区块

代码块

如果是段落上的一个函数或片段的代码可以用反引号把它包起来（`）

也可以用```包裹一段代码

链接

图片

表格

设置表格的对齐方式：

  • -: 设置内容和标题栏居右对齐。
  • :- 设置内容和标题栏居左对齐。
  • :-: 设置内容和标题栏居中对齐。
