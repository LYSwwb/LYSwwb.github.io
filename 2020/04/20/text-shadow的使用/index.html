一、CSS3 text-shadow属性介绍

text-shadow属性可以给页面文本添加阴影效果

语法

值 描述
h-shadow 必需.水平阴影的位置,允许负值.
v-shadow 必需.垂直阴影的位置.允许负值.
blur 可选值,模糊的距离
color 可选.阴影的颜色.

二、text-shadow能干点啥

效果

样式 效果
p {font-size: 80px; line-height: 800px; text-shadow: 10px 0 0 #600;}
p {font-size: 80px; line-height: 800px; text-shadow: -10px 0 0 #600;}
p {font-size: 80px; line-height: 800px; text-shadow: 0 10px 0 #600;}
p {font-size: 80px; line-height: 800px; text-shadow: 0 -10px 0 #600;}
p {font-size: 80px; line-height: 800px; text-shadow: 0 80px 10px #600;}
text-shadow: 0 -5px 4px #FF3,2px -10px 6px #fd3,-2px -15px 11px #f80,2px -25px 18px #f20;
text-shadow: -1px -1px white,1px 1px #333;
text-shadow: -1px 0 black,0 1px black,1px 0 black,0 -1px black; 
text-shadow属性的h-shadow控制的是阴影水平方向的位移,也就是阴影在文字的左右移动;
v-shadow控制阴影在垂直方向的位置,控制阴影上下移动.
text-shadow属性可以接受多个阴影效果,但是需要使用','隔开.
阴影效果按照先后顺序应用,阴影之间会出现覆盖,后定义的阴影将覆盖先定义的阴影.
阴影效果不会元素的尺寸,但是阴影可以超出元素边界.
