css选择器

发表于 2020-04-21 | 分类于 前端

一、为什么要用选择器

我们要为某个元素设置样式，那么首先要只要到相应的元素。在css中的选择器就能为找到相应的元素，提供了非常灵活的方案。

二、选择器的种类

css选择器共包含五大类：基本选择器、层级选择器、伪类选择器、伪元素选择器、属性选择器。

基础选择器

基础选择器种类

选择器 类型 功能
* 通配符选择器 选择文档中所有的html元素
elem 元素选择器 选择指定的类型的 elem 元素
#id id选择器 选择文档中id为指定 “ id “ 的元素
.class 类选择器 选择文档中class属性为指定 “ class ” 的所有元素
sel,sel 群组选取器 选择每一个选择器匹配的元素集合

基础选择器示例：

选择器 功能
* 匹配页面所有的html元素
div 匹配页面所有的div元素
#id 匹配页面id为“id”的元素
.calss 匹配页面class属性为“class”的所有元素的集合
div,p 匹配页面所有的div元素和p元素

基础选择器使用

选择文档所有的元素，清除元素的margin和padding;

层级选择器

层级选择器种类

选择器 类型 功能
el > el 子元素选择器 选择 el 元素的第一层子代 el 元素
el + el 相邻兄弟选择器 选择 el 元素的紧邻 el 元素
el ~ el 通用兄弟选择器 选择 el 元素的后面所有的 el 元素
el e 后代选择器 选择 el 元素的包含的所有的 e 元素

基础选择器示例：

选择器 功能
ul > li 选择页面中所有 ul 元素的第一层子代 li 元素
ul + div 选择页面中所有 ul 元素的紧邻的div元素
p ~ span 选择页面中所有 p 元素的后面的兄弟span元素
div p 匹配页面所有的div元素下的所有p元素

伪类选择器

伪类选择器种类

伪类选择器包含：动态伪类选择器、目标伪类选择器、语言伪类、UI元素状态伪类选择器、结构伪类选择器、否定伪类选择器。

动态伪类选择器
选择器 类型 功能
el:link 链接伪类选择器 匹配尚未访问时的 el 链接
el:visited 链接伪类选择器 匹配访问过的 el 链接
el:active 用户行为伪类选择器 用于链接或者按钮被激活时的状态
el:hover 用户行为伪类选择器 用于鼠标悬停在 el 上时的状态
el:focus 用户行为伪类选择器 当 el 获取焦点时的状态 
注：锚点伪类的顺序“ link  visited hover active ”;:hover在IE6仅支持a:hover.
目标伪类选择器
选择器 类型 功能
el:target 目标伪类选择器 匹配页面被URL指向的 el 元素 
注：必须有URL指向时才有效。
语言伪类选择器

语言伪类选择器允许在不同语言下，定义不同的规则，在多语言版本的网站中使用很方便。

UI元素状态伪类选择器
选择器 类型 功能
el:checked 选中状态伪类选择器 匹配页面被URL指向的 el 元素
el:enabled 启用状态伪类选择器 匹配所有被启用的 el 元素
el:disabled 禁用状态伪类选择器 匹配所有被禁用的 el 元素
结构伪类选择器
选择器 类型 功能
el:first-child 结构伪类选择器 父元素下的第一个 el 元素
el:last-child 结构伪类选择器 父元素下的最后一个 el 元素
el e:nth-child(n) 结构伪类选择器 匹配 el 元素下的第n个子元素 e
el e:nth-last-child(n) 结构伪类选择器 匹配 el 元素下的倒数第n个子元素 e
el e:nth-of-type(n) 结构伪类选择器 匹配 el 元素下的所有指定类型 e 的第n个元素
el e:nth-last-of-type(n) 结构伪类选择器 匹配 el 元素下的所有指定类型 e 的倒数第n个元素
el e:first-of-type(n) 结构伪类选择器 匹配 el 元素下的所有指定类型 e 的第1个元素
el e:last-of-type(n) 结构伪类选择器 匹配 el 元素下的所有指定类型 e 的倒数第1个元素
el:only-child 结构伪类选择器 匹配 元素 仅有一个子元素，且元素为 el
el:only-of-type 结构伪类选择器 匹配 元素 仅有一种类型的子元素，且元素类型为 el
el:empty 结构伪类选择器 匹配 元素 没有任何子元素，为空节点
结构伪类选择器：
选择器 功能
ul li:nth-child(1) 匹配 ul 元素下的第1个子元素 li
ul li:nth-child(odd) 匹配 ul 元素下的所有奇数子元素 li
ul li:nth-child(even) 匹配 ul 元素下的所有偶数子元素 li
ul li:first-child 匹配 ul 元素下的第1个子元素 li
ul li:last-child 匹配 ul 元素下的最后第1个子元素 li
否定伪类选择器
选择器 类型 功能
el:not(class) 否定伪类选择器 匹配页面中除了 class 外的所有的元素 el

伪元素选择器

伪元素选择器种类

伪元素选择器包含：::first-letter(第一个字母)、::first-line(第一行文本)、::before(创建一个内容但是内容不是DOM的一部分在元素前)、::after(创建一个内容但是内容不是DOM的一部分在元素后)、::selection(选中的文本)。

属性选择器

属性选择器种类

选择器 类型 功能
el[attr] 属性选择器 选择 el 元素且 el 具有 attr 属性
el[attr=val] 属性选择器 选择 el 元素且 el 的 attr 属性值为 val
el[attr =val] 属性选择器
el[attr~=val] 属性选择器 选择 el 元素且 el 的 attr 属性值为多个空格分隔的值，其中有一个值为val
el[attr*=val] 属性选择器 选择 el 元素且 el 的 attr 属性值存在val字符
el[attr^=val] 属性选择器 选择 el 元素且 el 的 attr 属性值是依 val 开头的字符串
el[attr$=val] 属性选择器 选择 el 元素且 el 的 attr 属性值是依 val 结尾的字符串
